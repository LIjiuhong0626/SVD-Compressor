# SVD Image Compression

### 项目初衷 
作者在学习 **PCA** 与 **SVD** 的过程中，觉得单纯从数学推导的角度理解“误差”或“低秩逼近”不够直观。

于是，我从 **矩阵近似** 的角度出发，开发了这个小工具。通过调整保留的奇异值数量 $k$，实时生成并对比不同的压缩图片，从而直观地感受 SVD 的数学魅力——即如何用极少的数据量（低秩矩阵）还原出高质量的图像信息。

---

### 
*   **直观展示**：拖动滑块即可改变 $k$ 值，实时查看图像重构效果。
*   **能量保留度计算**：定量显示当前 $k$ 值下图像“能量”（信息量）的理论保留比例。
*   **计算缓存机制**：引入 MD5 校验与 `.npz` 缓存，避免对同一张图片重复进行高昂的 SVD 运算。
*   **多格式支持**：支持导出为 JPG/PNG 格式，并自动规范化输出路径。

---

### 快速上手

#### 1. 克隆项目
```bash
git clone https://github.com/LIjiuhong0626/SVD-Compressor.git
cd SVD-Compressor
```

#### 2. 安装依赖
本项目推荐使用 [uv](https://docs.astral.sh/uv/) 进行管理：
```bash
uv sync
```
*或者使用传统 pip：*
```bash
pip install numpy pillow pillow-heif
```

#### 3. 运行程序
```bash
python main.py
```

---

### 使用说明 
1.  **Setting 页面**：点击 `Select Image` 选择图片，点击 `Compute SVD` 进行矩阵分解（首次计算需耗时几秒）。
2.  **调整 $k$ 值**：拖动滑块或输入数字，观察 **Energy Retained** 和预览图的变化。
3.  **Export 页面**：设置文件名和保存路径，将满意的压缩结果导出。
